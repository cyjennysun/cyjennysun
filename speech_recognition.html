Introduction

This project is part of the JavaScript 30-day Challenge. It incorporated Web Speech API - speech recognition. Users can speak to the microphone, and a text string will be returned. I also added a "copy to clipboard" functionality that enables users to copy the text with just one click. Speech recognition is only supprted in Chrome 33+ , Firefox 44+, and Firefox OS 2.5+.
