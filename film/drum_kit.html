Drum Kit

Introduction

I did this project for JavaScript 30-day Challenge. It incorporated audio elements and interactive interface. Each key produces different sounds. Users can press the corresponded keys on the keyboard to play the drum kit. This game is designed for desktop only.

Languages & Skills

  • Vanilla JavaScript
  • Audio elements manipulation + keyCodes
