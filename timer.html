Countdown Timer

View Project

Introduction

This is a personal project. I aimed to explore various event listeners in jQuery. Users can set a timer by selecting minutes and seconds, with a maximum limit of one hour. Pause and reset functions are supported. When time is up, a mellow bell tone will start ringing to notify users.
