Introduction

This project is part of the JavaScript 30-day Challenge. It incorporated Web Speech API - speech synthesis. Users can turn their text into speech by selecting a voice and a language in the dropdown menu, and typing any desired content in the text box.

  • Vanilla JavaScript

  • Browser support - feature detection
